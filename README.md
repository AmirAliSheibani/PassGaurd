# PassGuard

PassGuard is a secure password manager built with Django. It allows users to store and manage credentials inside an encrypted vault protected by a master password.

## Features (In Progress)
- Secure credential storage  
- PostgreSQL database integration  
- Environment-based configuration  
- Django full-stack architecture  

## Status
This project is currently under active development. Features and structure may change frequently.

## Tech Stack
- Python  
- Django  
- PostgreSQL  
- Django Templates
